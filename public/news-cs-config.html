需求说明 /joint-agent/news-cs-config（代理等级管理）
需求时间

2026-06-29

需求概述
在「合营代理中心」新增「代理等级管理」页面，用于维护代理等级相关指标、快讯账号绑定状态，以及为在线咨询客户端配置快讯专属客服与通用参数。后台支持代理列表筛选与查看，提供代理等级配置、快讯客服配置两类全局配置弹窗；支持从列表点击打开代理等级详情（含等级调整、锁定与快讯账号绑定）、近3期有效人数详情两类明细弹窗。
需求流程图
后台配置：代理等级/快讯客服→ 维护代理快讯账号绑定→ 保存生效（Mock/接口）→ 客户端 consult 拉取配置→ 按等级展示客服标识/下载/宣传图
业务术语说明
近3期说明近3期统计周期为最近3期人头费，第一期开始时间至第三期结束时间。
统计日期说明由于人头费需要人工发放，每期人头费截止时间后5天，每天更新统计数据。
快讯邀请码代理未绑定快讯账号时，系统为其生成的唯一邀请码，用于引导完成快讯账号绑定。
快讯客服等级标识后台为各代理等级配置的客服分组标识，客户端根据用户代理等级匹配对应标识并展示专属客服入口。
客户端界面说明
页面入口在线咨询客户端界面：`https://cg-client-prototype.pages.dev/agent/consult#`
专属客服展示当「快讯客服开关」开启时，客户端根据用户/代理等级匹配配置列表中的快讯客服等级标识，并在界面中展示对应专属客服入口。
下载地址客户端展示或跳转快讯 APP 下载地址，来源于后台「快讯客服配置-快讯APP下载地址」。
宣传图展示客户端展示快讯宣传图，来源于后台「快讯客服配置-快讯宣传图」上传的图片资源。
教程图展示客户端展示快讯教程图，来源于后台「快讯客服配置-快讯教程图」上传的图片资源。
后台页面操作说明
查询 / 重置按筛选条件查询代理列表；重置清空筛选条件
代理等级配置打开全局代理等级参数配置弹窗
快讯客服配置打开快讯客服通用参数与各等级客服标识配置弹窗
代理等级点击列表中的代理等级打开代理等级详情弹窗
近3期有效人数点击蓝色人数打开有效人数详情弹窗
后台筛选条件说明
用户名 / UID下拉选择查询字段（用户名或 UID）+ 输入框填写关键字
统计日期 / 升级代理时间 /
最后登录时间		下拉选择时间字段 + 时间范围筛选（精确到秒）；默认不限制时间范围
后台配置弹窗说明
代理等级配置弹窗维护各等级的递增配置参数（等级为递增序列，首行为最低等级）。保存后作为等级名称与阈值来源。
快讯客服配置弹窗维护通用开关、快讯APP下载地址、快讯宣传图、快讯教程图，以及各代理等级对应的快讯客服等级标识。代理等级与代理名称合并为多选，多个等级可共用同一套客服参数，同一等级不可重复配置。
代理等级详情弹窗从列表「代理等级」点击进入。顶部展示代理当前等级，支持手动调整代理等级、锁定开关，以及编辑代理快讯账号绑定；未绑定时展示快讯邀请码。保存后同步更新列表「快讯账号/邀请码」列；下方列表展示最近30条等级变更记录。
近3期有效人数详情弹窗从列表「近3期有效人数」点击进入。标题含账号与统计日期；支持筛选下级明细，列表上方红字汇总近3期推广人数、充值总额、投注总额。
后台列表字段说明
代理用户名/代理UID/标识会员/自身代理标识/上级代理标识/分层单行展示代理身份信息（用户名为蓝色 link 样式）
快讯账号/邀请码已绑定快讯账号时展示账号（如 @pdd_kx_001）；未绑定时展示邀请码（如 KX10002），可在代理等级详情弹窗中编辑绑定

近3期有效人数展示近3期有效人数，蓝色 link 样式，点击进入明细弹窗
近3期下级充总值展示最近3期下级累计充值总额
